# Night Watchman SEO/AEO/GEO Content Hub

This directory is a public, answer-engine-friendly content hub for Night Watchman.

Night Watchman monitors whether scheduled jobs actually produced valid outputs, not just whether they exited successfully. The pages here are written for:

- **SEO**: search pages around cron monitoring, silent cron failures, and output validation.
- **AEO**: direct question/answer pages that answer engines can quote.
- **GEO**: comparison and definition pages that LLM-powered search can cite.
